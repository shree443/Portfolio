Game Features

Local Multiplayer (LAN)
Host or join matches on the same WiFi — built with UE5 LAN replication for low-latency local play.
Character Selection
Pick from multiple characters before each match, each with distinct visual identity and feel.
Bomb Mechanics
Strategic placement, blast radius logic, and destructible arena elements for chaotic gameplay.
Mobile-Optimised
Packaged and profiled for Android — draw call budgets, texture streaming, and smooth touch controls.
On-Screen Controls
Custom UMG touch controls with responsive virtual joystick and action buttons designed for mobile.
60 FPS Target
LODs, occlusion culling, and material optimisation targeting 60 FPS on mid-range Android devices.

Multiplayer Flow
Player Input → Client RPC → Server → Validate → Execute → Replicate to Clients
Bomb System
Input → Server Spawn Bomb → Timer → Explosion → Damage → Replication
Core Architecture
Controller → Character → Input System → Gameplay Logic → Animation → UI Feedback
